# Comp331-Final-Project

This project analyzes the UCI Wine Quality datasets (Red and White) to evaluate data quality 
  (focusing on validity, completeness and consistency) and perform ETL cleaning.

## Project Contents
 - 'wine_quality_etl_analysis.ipynb': Jupyter notebook with full analysis
 - 'combined_cleaned_dataset.csv': Cleaned and combined dataset ready for analysis
 - (change this to wine_quality_cleaned_dataset)
 - 'duplicates.csv': Table showing all duplicate data which is then deleted.
 - 'duplicates_oer_wine.csv': Table showing number of duplicate data catagorized by wine type
 - 'EMPTY_invalid_values': Seperate table for each invalid value broken down by category

## Dataset
Source: [UCI Wine Quality Dataset]: https://archive.ics.uci.edu/dataset/186/wine+quality
